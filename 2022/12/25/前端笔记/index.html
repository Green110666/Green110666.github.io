网页是构成网站的基本元素，它通常由图片、链接等元素组成。我们通常看到的网页，常见以.html或.htm后缀结尾的文件。因此将其称为HTML文件，这是超文本标记语言，是用来描述网页的一种语言，是标签语言不是编程语言。

浏览器内核：负责读取网页内容，整理讯息，计算网页的显示方式并显示页面。例：Webkit / Blink内核

Web标准是由WBC（万维网联盟）组织和其他…制定的一系列标准的集合。

选择器

CSS 选择器用于“查找”（或选取）要设置样式的 HTML 元素。

我们可以将 CSS 选择器分为五类：

  • 简单选择器（根据名称、id、类来选取元素）
  • 组合器选择器（根据它们之间的特定关系来选取元素）
  • 伪类选择器（根据特定状态选取元素）
  • 伪元素选择器（选取元素的一部分并设置其样式）
  • 属性选择器（根据属性或属性值来选取元素）
